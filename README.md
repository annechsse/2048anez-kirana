# Game 2048 - Anezka Kirana

Anggota:
- Anezka
- Kirana

## Cara Memainkan Game
1. Buka file `index.html` menggunakan browser (Chrome/Edge).
2. Gunakan tombol arah pada keyboard untuk menggerakkan angka:
   - Atas (↑)
   - Bawah (↓)
   - Kiri (←)
   - Kanan (→)
3. Gabungkan angka yang sama hingga mencapai angka 2048.
4. Game selesai jika papan penuh dan tidak ada lagi langkah yang bisa dilakukan.
